## Info
<a href="https://www.acmicpc.net/problem/11060" rel="nofollow">11060 점프 점프</a>

## 풀이
`dp[i]`는 `i번째에서 최소 점프 횟수`로 정의한다. `A[i]`만큼 점프를 할 수 있으므로 `i+1<=j<=i+j`만큼 반복하며 `dp[j]`를 갱신한다. 

아직 어떠한 값도 들어가있지 않은 상황이라면 기존 횟수에 1을 더한다(`dp[i]+1`). 이는 `기존 횟수`에 `한 번` 더 점프했다는 뜻이다.

이미 어떠한 값이 들어가있는 상황이라면 기존 `dp[j]`와 `dp[i]+1` 중에서 최솟값을 선택한다. 

정답은 `dp[N]`에 담기도록 해놓았는데, 대신 배열에서 1을 제외하고 `2<=i<=N` 범위에서 중간에 0이 있다면 점프가 중간에 끊긴 것이므로 끝까지 갈 수 없다는 뜻이다. 따라서 -1을 출력한다.


## 마무리
다양한 `Dynamic Programming` 문제를 풀어볼 수 있는 시간이었다.
