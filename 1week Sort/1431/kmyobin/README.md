## ?? 풀이
정렬 기준이 총 3가지이다.
1. 길이가 짧은 것이 앞으로
2. 길이가 동일한 경우 모든 자리수의 숫자의 합이 더 작은 것이 앞으로
3. 위의 두 경우에도 비교할 수 없을 때는 사전순 비교

이 때 3번에서 사전순 비교는 아스키 코드로 비교할 수 있다. (0 = 48, A = 65)

정렬은 `bubble sort`형식과 swap 함수를 이용하였다.

길이가 동일할 때 숫자의 합을 구한다. 이 때 숫자까지 같다면 3번 규칙에 따라 사전순으로 정렬한다.

시리얼 번호를 아스키 코드(int)로 받고, 앞에 있는 아스키 코드가 더 크면 swap 후 break, 앞에 있는 아스키 코드가 더 작으면 바로 break를 걸어서 빠져나온다.

## 마무리
3번 조건을 설정하는데 꽤 오랜 시간이 걸렸다. 나중에는 tuple에 대해서도 배워서 적용해보아야겠다.
