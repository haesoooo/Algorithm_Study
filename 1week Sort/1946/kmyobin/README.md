## ?풀이
핵심 조건은 서류심사 성적과 면접시험 성적 중 하나라도 다른 지원자보다 떨어지면 탈락이라는 것이다.

처음에는 서류심사, 면접시험에 관해 총 2개의 분야를 정렬해야되나 생각했는데, 그렇게 되면 시간 복잡도가 커질 것 같아서 한 분야만 정렬한 후 처리하기로 했다.

따라서 quicksort 함수 부분에서 서류심사 기준으로 오름차순 정렬을 진행하였다. 그 다음 정렬된 벡터에서 면접시험 점수를 비교하였다.

`compare_interview` 함수에서 제일 낮은 성적을 `A[0]`의 면접시험 성적으로 받아놓고, `index 1`부터 제일 낮은 순위보다 더 낮다면 그 사람을 탈락시키는 의미에서 `cnt--`을 진행한다. 만약 `low_grade`보다 높은 순위라면 순위를 갱신시켜야 하므로 `low_grade`에 `A[i].second`를 할당한다.

## 마무리
직접 sort 알고리즘을 구현하기 보단 `algorithm` 헤더에 있는 것을 이용하여 세부 사항을 설정하는 방향으로 코드를 구성해야겠다.
