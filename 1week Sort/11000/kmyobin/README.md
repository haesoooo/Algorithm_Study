## ?풀이
`우선순위 큐(Priority Queue)`를 이용한 문제이다. 강의실을 새로 여는 조건은, `현재 있는 강의실이 모두 사용 중일 때`이다.

시작 시간과 끝나는 시간을 벡터에 담고, 시작 시간을 기준으로 오름차순 정렬한다.

다음으로 우선순위 큐에 첫 끝나는 시간을 push한다.
`A[1]`부터 접근하며 해당 시작 시간이 우선순위 큐에 있는 `top()`보다 작으면 강의가 끝나지 않았다는 것으로, 강의실을 추가해야한다. 이는 `i번째 인덱스의 끝나는 시간`을 우선순위 큐에 push해야 한다는 뜻이다.

만약 시작 시간이 우선순위 큐에 있는 `top()`보다 크면 강의가 끝났다는 뜻이므로 이어서 쓸 수 있다. 따라서 우선순위 큐에 있는 값을 pop하고 새로운 강의의 끝나는 시간을 push한다.

이렇게 되면 우선순위 큐의 크기가 강의실의 개수와 같게 된다.


## 마무리
우선순위 큐에 대해 알게 된 시간이었다. 생각보다 많이 나오는 개념이라 더 공부해야겠다고 생각하였다. 우선순위 큐 + Greedy 조합에 대해 더 알게 되었다.
