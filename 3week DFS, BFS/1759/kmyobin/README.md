## Info
<a href="https://www.acmicpc.net/problem/1759" rel="nofollow">1260 암호 만들기</a>

## 풀이
`조합`을 떠올려야 하는 문제이다. 알파벳이 암호에서 증가하는 순서여야 하므로 받은 그래프를 `오름차순 정렬`한 후 문제를 풀었다.

`백트래킹`을 이용하였는데, 이는 보통 `DFS`와 비슷한 개념이다. 대신 `백트래킹`은 `해가 아니라면 다시 돌아온다`. 이는 `vector`에 해당 좌표를 넣고, 빼는 것으로 구현되어있다.

최소 1개의 모음과 최소 2개의 자음으로 구성되어야 하므로 함수에서 새롭게 모음과 자음을 갱신하여 파라미터로 넣어주었다.

## 마무리
`백트래킹` 구현 방법에 대해 알게 되었고, `DFS`와 살짝 다른 것을 깨달았다.
