## Info
<a href="https://www.acmicpc.net/problem/2606" rel="nofollow">2606 바이러스</a>

## 풀이
`DFS`로 풀었다.

1번 컴퓨터부터 시작한다고 문제에 주어져있다. 최대한 깊게 들어간 다음 빠져나오는 것이 좋을 것이라 생각하여 `DFS` 풀이를 선택하였다.

`dfs` 함수의 파라미터 `r`은 행을 의미하며, 해당 행을 탐색했다면 `visited[r]`를 `true`로 바꿔준다.

새로 방문한 행일 때마다 `answer` 값을 증가시켜 바이러스에 걸리게 되는 컴퓨터 수를 표현하였다.

## 마무리
`DFS` 코드를 직접 구현해보는 시간이었다. 무조건 행, 열 파라미터를 넘겨야 된다고 생각했는데 아니었다.
