## Info
<a href="https://www.acmicpc.net/problem/1260" rel="nofollow">1260 DFS와 BFS </a>

## 풀이
전형적인 `DFS`와 `BFS` 탐색 문제이다.

`DFS`는 `깊이우선탐색`으로 최대한 깊이 내려간 뒤, 더 이상 깊이 갈 곳이 없다면 옆으로 이동하는 방법이다. 검색 속도는 `BFS`에 비해 느리다. `stack` 또는 `재귀`로 구현할 수 있다.

`BFS`는 `너비우선탐색`으로 최대한 넓게 이동한 뒤, 더 이상 갈 수 없다면 아래로 이동하는 방법이다. `queue`로 구현할 수 있다. 보통 최단 거리를 구해야 하는 문제일 때 `BFS`가 유리하다.

문제에서 특이한 조건이 있다면 방문 정점이 여러 개인 경우, 정점 번호가 작은 것을 먼저 방문한다는 점이다. 따라서 그래프를 `오름차순 정렬`하여 작은 것 먼저 접근하게 하였다.

## 마무리
`DFS`와 `BFS`의 정의와 구현 방법에 대해 복기할 수 있는 시간이었다.
