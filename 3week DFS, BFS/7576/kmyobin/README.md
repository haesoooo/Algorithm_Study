## Info
<a href="https://www.acmicpc.net/problem/7576" rel="nofollow">7576 토마토</a>

## 풀이
`BFS`을 활용하여 풀었다.

대신 순차적으로 `for문`을 이용하여 접근하면 여러 개의 익은 토마토가 있을 때에도 한 개의 토마토를 기준으로 `bfs`가 실행되므로 `queue`에 익은 토마토를 미리 담고 시작하였다. 값이 증가할 때마다 `answer`값을 할당하였다.

정답을 쓸 때는 조건문을 이용하였다. 안 익은 토마토가 있는 경우 `-1`을 출력하고, `check()`에서 `for문`을 모두 돌았다면 모두 익은 토마토이므로 정상적으로 `answer`를 출력하였다.

## 마무리
마지막 답을 출력하는 과정에서 시간이 많이 걸렸다.
