## 풀이
`multimap`을 이용하여 풀었다. `multimap`은 key값의 중복이 가능하며, 이외는 `map`의 성질과 동일하다. 이 문제에서 `key`값은 `의상 종류`로 지정하였고, `value`값은 `의상 이름`으로 지정하였다. `key`값은 배열에 한 번 더 담아서 `key`값을 찾는 데에 수월하게 하였다. `의상 종류 + 1`를 곱한 다음에 `1`을 빼주었는데, 이는 `아무것도 안입는다`는 뜻의 1을 제거한 것이다. 아무것도 안입으면 안된다는 규칙이 문제에 쓰여있었기 때문이다.


## 마무리
`vector`와 `map`을 많이 써보지 않았는데, 앞으로 공부를 더 많이 하여 함수 사용에 능숙해져야겠다.
