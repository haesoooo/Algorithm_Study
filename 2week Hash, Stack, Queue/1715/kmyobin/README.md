## Info
<a href="https://www.acmicpc.net/problem/1715" rel="nofollow">1715 카드 정렬하기</a>

## 풀이
`우선순위 큐`를 이용하여 문제를 풀었다. 처음에는 기존에 더한 것을 곱하기 2하고, 그것을 다시 더하며 곱하기 2를 진행하는 로직을 짰었는데 이 방법이 항상 최솟값을 가지는 것이 아님을 깨닫고 풀이 방식을 바꾸었다. 우선순위 큐는 값이 들어올 때마다 정렬이 되므로 오름차순 정렬을 하여 업데이트되는 값을 넣어주었다. `기존값 + (기존값 + 새로운 값)`과 `기존값 + (새로운 2개 값의 합)`을 비교하여 더 적은 것을 배열에 담는 방식으로 문제를 해결했다. 그리고 배열에 담긴 값들을 모두 더해주어 정답을 산출하였다.


## 마무리
우선순위 큐가 나오는 문제가 많아서 더 세밀한 공부가 필요한 것 같다.
