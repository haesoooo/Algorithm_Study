## Info

## 풀이
`stack`을 사용하는 문제이다. 이 문제에서 괄호 처리를 어떻게 할 지 많은 고민을 했다. 실제로 예제를 풀어보며 로직을 구성하였다. 

문자가 나올 때는 바로 출력하고, `+`, `-`일 때는 우선순위가 낮은 부호이므로 `c.top()`이 `*`, `/`인지, 아니면 `+`, `-`인지에 따라 출력 방식을 다르게 했다. `c.top()`값이 우선순위가 더 높은 경우 `stack`에 있는 값을 출력하였고, 우선순위가 동일한 경우는 한 번만 `stack`에 있는 값을 출력하였다. `)`일 때는 `(`이 나올 때까지 계속 pop해주었다.

## 마무리
`stack`을 이용한 괄호 문제가 다양한 것 같다. 더 자세히 공부해야겠다.
