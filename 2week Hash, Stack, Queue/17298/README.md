## Info
<a href="https://www.acmicpc.net/problem/17298" rel="nofollow">17298 오큰수</a>

## 풀이
시간제한이 `1초`이므로 `O(N^2)`로 풀면 시간초과가 발생한다. 따라서 `O(N)`을 만족하며 탐색해야한다.

오른쪽에 있는 숫자 중 제일 왼쪽에 있는 것을 뽑아야하므로 `stack`을 이용하였다. 비어있으면 `B[i]` 오른쪽에 `B[i]`보다 큰 수가 없다는 뜻이므로 `-1`을 저장하고, `B[i]`보다 `A.top()`이 더 크면 오른쪽에 있으면서 `B[i]` 제일 왼쪽에 있는 수를 뜻하므로 저장한다.

## 마무리
`stack`에 대해 더 공부해야겠다. 시간 복잡도를 생각하며 문제를 풀어야 허투로 시간을 쓰지 않는 것 같다.
